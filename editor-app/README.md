# Image Editor

A simple , fast and feature rich Image editor created by Abhay, a full stack dev, who is frusted by not being able to find a good solution for his problem of quickly editing his screenshot in his ubuntu desktop enviroment.

### Features (for v1)

- [ ] Super fast load time : Use service worker and offline cache first approach
- [ ] Clipboard image paste : Ability to fast images from clipboard to canvas for quick editing
- [ ] Basic Editor : Text, brushes and Images for now.
- [ ] Export : Quickly export the canvas to downloadable Image
