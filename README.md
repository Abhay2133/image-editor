# Image Editor

A Simple yet powerfull and speedy image editor for web<br/>
**Motivation** : Tired of no built-in quick image editor in ubuntu

### Version 1 Features

1. Canvas  : To draw and apply overlays
2. Paste   : You can paste copied images from clipboard
3. Save    : Save image in major image formats
4. Offline : Make it a PWA
5. Resize  : rectangular Crop and resize image as desired

### Version 2 Features

1. Effects : adding color filter
2. Masking : implement image masks (cirular crops and shape crops also)
3. Layers  : add a layer system 
4. GRID    : Grid System for arranging things with snapping behavior
